# Academic website starter (Quarto + GitHub Pages)

This is a minimal personal academic website starter built with Quarto.

## 1. Install Quarto
Download Quarto from the official website.

## 2. Preview locally
In the project folder, run:

```bash
quarto preview
```

## 3. Render the site
```bash
quarto render
```

This will create the rendered website in the `docs/` folder.

## 4. Publish on GitHub Pages
Create a GitHub repository, push this project, then configure GitHub Pages to publish from the `docs/` folder on the main branch.

## 5. Replace placeholders
Edit:
- your name
- bio
- links
- publications
- CV PDF
- GitHub / ORCID / ADS / Scholar links
